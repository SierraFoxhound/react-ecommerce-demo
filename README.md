# React E-Commerce Demo (Sage & Stone)

A demonstration e-commerce storefront built with React (Vite).

This mock store showcases product listing, cart functionality, and user interaction with a modern React frontend. The backend is in a separate repository(https://github.com/SierraFoxhound/ecommerce-backend).

## About

React E-Commerce Demo is a sample web app that simulates an online store.
It allows users to:

-Browse products

-Add items to a cart

-Remove items

-See dynamic cart totals
